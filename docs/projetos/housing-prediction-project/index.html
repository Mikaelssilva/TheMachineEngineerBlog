Seu Primeiro Projeto de Machine Learning

Objetivo: Construir um modelo que prevê preços de casas
Tempo: 30-45 minutos
Nível: Iniciante (Python básico)
Resultado: Modelo com 80% de precisão!

Neste projeto, você vai aprender:

Análise de Dados

Aprenda a interpretar dados, encontrar padrões e entender as métricas que influenciam o modelo.

Manipulação de Dados

Use bibliotecas como Pandas e Numpy para limpar, transformar e preparar seus dados para modelagem.

Seleção de Modelos

Descubra como escolher o melhor modelo para o seu problema, diferenciando entre modelos de regressão e classificação.

3. Preparando os Dados

Separando características e alvo

Agora que os dados estão carregados, vamos separá-los em características (X) e alvo (y). As características são as variáveis que o nosso modelo usará para aprender, e o alvo é o que queremos prever.

x = dados.drop('target', axis=1)
y = dados['target']

Dividindo em treino e teste

O próximo passo é dividir os dados em conjuntos de treino e teste. O modelo vai aprender usando os dados de treino e será avaliado usando os dados de teste. Isso garante que ele consiga generalizar para novos dados que ele nunca viu antes.

np.random.seed(42)

from sklearn.model_selection import train_test_split
x_train, x_test, y_train, y_test = train_test_split(x, y, test_size=0.2)
Divisão dos Dados

O parâmetro test_size serve para determinar o tamanho da nossa amostra de teste. É convenção deixar 80% dos dados para treinar e os demais 20% para teste.

4. Selecionando o Modelo

from sklearn.ensemble import RandomForestRegressor

# Instanciar o modelo
modelo = RandomForestRegressor(random_state=42)
Regressão vs Classificação
  • Regressão: Quando você quer prever um número (preço, idade, temperatura)
  • Classificação: Quando você quer prever uma categoria (spam/não spam, gato/cachorro)

Como queremos prever preços (números), nosso problema é de regressão!

5. Treinando o Modelo

Treinar nosso modelo é super simples. Basta usar a função fit():

modelo.fit(x_train, y_train)

Testando com uma casa nova

Vamos criar dados de uma casa hipotética para ver como o modelo se comporta.

Formato dos Dados

Os dados que você passar para o modelo terão que estar no mesmo formato que você usou para treinar. No nosso caso, utilizamos um DataFrame do Pandas.

# Dados de uma casa hipotética
dados_nova_casa = {
    'MedInc': [5.0],      # Renda mediana: $50k
    'HouseAge': [30.0],   # Casa com 30 anos
    'AveRooms': [6.0],    # 6 cômodos em média
    'AveBedrms': [1.0],   # 1 quarto por casa
    'Population': [1000.0], # População da região
    'AveOccup': [3.0],    # 3 pessoas por casa
    'Latitude': [34.0],   # Localização
    'Longitude': [-118.0]
}

nova_casa = pd.DataFrame(dados_nova_casa)
preco_previsto = modelo.predict(nova_casa)

print(f"O preço previsto para a nova casa é: ${preco_previsto[0]:.2f}")
print(f"Isso equivale a aproximadamente: ${preco_previsto[0]*100:.0f}k")
O preço previsto para a nova casa é: $2.30
Isso equivale a aproximadamente: $230k

7. Avaliando o Modelo

Para avaliar nosso modelo usaremos a função score():

precisao = modelo.score(x_test, y_test)
print(f" Precisão do modelo: {precisao:.3f}")
print(f"Isso significa: {precisao*100:.1f}% de acerto!")
 Precisão do modelo: 0.805
Isso significa: 80.5% de acerto!
